## 导航栏升级说明

### 新增功能：应用模式切换

导航栏现在支持在**雷达图**和**时间轴**两个工具模式之间切换。

#### 视觉设计

参考 Omada/Ubiquiti 风格：
- 导航栏背景色：绿色系（浅色 `#4a9d7f`，深色 `#2a5a4a`）
- 选中的 Tab：背景色与主界面一致（浅色 `#f5f5f5`，深色 `#0d0d0d`）
- 未选中的 Tab：透明背景，白色半透明文字
- 悬停效果：白色半透明背景
- 圆角效果：顶部圆角 8px，模拟打开的页面

#### 技术实现

**新增状态** (`uiStore.ts`):
```typescript
appMode: 'radar' | 'timeline'  // 当前应用模式
setAppMode: (mode: AppMode) => void
```

**导航栏组件** (`Navbar/index.tsx`):
- 新增模式切换 Tab（雷达图、时间轴）
- 图标：RadarChartOutlined、HistoryOutlined
- 支持点击切换模式

**样式** (`Navbar.module.css`):
- 导航栏使用绿色渐变背景
- 选中 Tab 自动匹配主界面背景色
- 响应式圆角和阴影效果
